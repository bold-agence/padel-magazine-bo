Classements

Créez un classement (ex. Hommes), puis importez un fichier Excel (.xlsx / .xls) : les lignes existantes sont remplacées à chaque import.

En-têtes reconnus (1re ligne) : rank, player / joueur, pointsNow, tournaments / tournois, previousRank, pointsPrev, rankDelta, pointsDelta — ou 8 colonnes sans en-tête dans cet ordre. Les cellules vides ou « - » sont acceptées : nombres manquants → 0, deltas vides → « - », rang (Pos) vide → numéro de ligne dans le fichier.
